# 📊 HR Data Analytics Dashboard

![HDB.png)

## 📌 Project Overview

The **HR Data Analytics Dashboard** analyzes employee attendance data to provide meaningful insights into workforce behavior. The dashboard helps HR teams monitor **employee presence, work-from-home trends, and sick leave patterns** across different months.

It is designed so that when the **dataset is updated, the dashboard automatically refreshes and updates all insights**, enabling HR teams to make quick data-driven decisions.

---

# 🧩 Problem Statement

HR teams often struggle to track employee attendance patterns and understand workforce behavior. Without proper analytics tools, it becomes difficult to identify:

* Work-from-home trends
* Employee presence rates
* Sick leave patterns
* Monthly attendance variations

This project aims to solve this problem by creating an **interactive HR analytics dashboard** that provides clear and actionable insights into employee attendance data.

---

# 🎯 Project Objectives

* Analyze employee attendance data
* Track **Presence Percentage**
* Monitor **Work From Home (WFH) Percentage**
* Analyze **Sick Leave Trends**
* Compare attendance across **April, May, and June**
* Create a **dynamic dashboard that updates automatically when the dataset changes**

---

# 🗂 Dataset Information

The dataset contains employee attendance records for:

* April
* May
* June

### Dataset Fields

* Employee ID
* Date
* Attendance Status
* Work From Home (WFH)
* Sick Leave (SL)
* Present in Office

---

# 📈 Key Metrics

### 🟢 Presence Percentage

Shows the percentage of employees present in the office.

### 🏠 Work From Home Percentage

Displays the percentage of employees working remotely.

### 🤒 Sick Leave Percentage

Shows the percentage of employees taking sick leave.

### 📅 Monthly Attendance Analysis

Compares attendance trends across **April, May, and June**.

### ⏰ Employee Work Patterns

Analyzes when employees start and end their workday.
